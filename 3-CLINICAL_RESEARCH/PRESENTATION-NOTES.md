# Clinical Research at the Cutting Edge — Presentation Notes

**11:15–12:00 | 45 min | After SARF Intro + Q&A, before lunch**

---

## I. Connecting to This Morning (5 min)

- Without measurement, can't tell when we're making it worse
- This morning: practices that harmed, professions swept up, nobody asked "what if this is making it worse?"
- Cambridge-Somerville: caring intervention → more crime, more addiction, earlier death
- *"How would we know?"* — this presentation answers that

## II. The Scaling Problem (10 min)

- Big data vs. small data — our problem is small data (rich, requires interpretation)
- Coding problem: someone has to read cases, identify SARF shifts, record on timeline
- Why it never happened: requires skilled coders, doesn't scale, nobody's day job
- What changed: machine automation makes intelligent inference possible at scale
- *"We built this."*

## III. Domain-Expert Ground Truth (10 min)

- Ground truth = what domain experts agreed on (train → blind code → measure agreement)
- Machines don't decide what's correct. Humans do. Every output reviewable.
- Infrastructure: web-based IRR system — case management, blind coding, agreement stats
- **The killer combo:** All AI needs is IRR. Agreement-verified coding = ground truth. *"IRR + AI = the breakthrough."*

## IV. Discoveries (10 min)

- Synthetic clinical discussions — realistic, SARF-codeable, controlled training environment
- **Mutual discovery** — can't survey for this like a knee problem. Therapist must explore *with* client or data stalls.
- **Synthetic data trains faster** — working group learned coding faster on synthetic than real cases
- **Responsibility pattern** — two interviews, person sees: constant yes, no filter, anxiety up, no downs. *"Just seeing the data organized was the breakthrough."*

## V. What Becomes Possible (5 min)

- **Era 1** (now): AI learns from humans. **Era 2** (follows): AI discovers patterns humans can't see
- If this works: certification, standardized training, aggregate research, vertically integrated tools
- Vertical alignment: one coding form → comparison, stats, replication, ground truth
- Getting involved → *"More on this at day's end."*

## VI. Honest State + Close (5 min)

- Early stage. First IRR meeting next week. Synthetic data working. Real recordings exploring.
- Human remains central: ground truth set by experts, software replicates, everything reviewable
- Tools: Pro (released), Personal (in dev), IRR (in use)
- *"Technology scales. Humans validate."*

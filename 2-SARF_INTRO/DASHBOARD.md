## Purpose

Introduce the SARF data model as a minimal, testable framework for collecting data on the relationship-symptom connection. This follows "Why Family?" which established the scientific problems.

## Core Message

SARF (Symptom, Anxiety, Relationship, Functioning) is the minimum framework needed to start collecting data on how relationships impact symptoms. The etiological pipeline is R → A → S, with F as the independent clinical variable.

## Key Concepts

| Variable | Definition | Value Space |
|----------|------------|-------------|
| **S** (Symptom) | Problems that bring people in | up, down, same |
| **A** (Anxiety) | Autonomic threat response to perceived uncertainty + urgency | up, down, same |
| **R** (Relationship) | Automatic moves toward/away from others | conflict, distance, reciprocal, child-focus, cutoff, triangle (in/out), define-self |
| **F** (Functioning) | Balance of emotion/thinking toward goals | up, down, same |
